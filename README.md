# Hangman Game 🎮

A console-based Hangman game implemented using object-oriented programming principles in [Java/Python/etc.].

## ✨ Features

- Difficulty Levels: Easy, Medium, Hard (10, 8, 6 max wrong attempts respectively)
- Word loading from external `words.txt` file, categorized by difficulty
- Input validation and dynamic feedback (correct/incorrect guesses)
- Full win/loss detection and game loop
- OOP concepts: inheritance, interfaces, polymorphism, and abstraction

## 📁 Project Structure

- `src/`: Source code implementing game logic
- `words.txt`: Word list with difficulty categories
- `ClassDiagram.pdf`: UML class structure of the system
- `SequenceDiagram.pdf`: Sample object interaction diagram

## 🛠 Technologies

- Language: Java/Python
- Concepts: OOP, File I/O, UML Design
- No GUI – purely console based
